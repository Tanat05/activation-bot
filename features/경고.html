⚠️ 경고 기능

서버 내 유저에게 경고를 부여하고 관리하는 기능입니다.

명령어

명령어 설명
!경고 [유저] [사유] 멘션한 유저에게 경고를 1회 부여합니다.
!경고확인 [유저] 멘션한 유저의 현재 경고 횟수를 확인합니다.
!경고차감 [유저] 멘션한 유저의 경고를 1회 차감합니다.

설정 방법

  1. 경고 기능은 별도의 설정 없이 바로 사용할 수 있습니다.
  2. 관리자 권한이 있는 유저만 명령어를 사용할 수 있습니다.
  3. 경고가 부여되면 지정된 로그 채널에 자동으로 기록됩니다. (로그 기능 설정 필요)
