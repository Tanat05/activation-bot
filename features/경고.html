명령어

명령어 설명
/경고 부여 유저 사유 선택한 유저에게 경고를 1회 부여합니다.
사유의 경우 처벌 로그채널이 설정되어있다면 로그에 사유가 같이 표기됩니다.
/경고 제거 유저 선택한 유저의 경고가 1회 제거됩니다.
/경고 자동제거 [역할] [경고수] [시간] [역할]를 가진 유저의 경고를 [시간]마다 [경고수]를 차감합니다
/경고 자동제거목록 현재 설정된 경고 자동제거의 목록을 확인합니다
/경고 자동제거해제 [역할] [역할]에 설정된 경고 자동제거 기능을 해제합니다
/경고 제거 [유저] 선택한 [유저]의 경고를 1회 차감합니다.
명령어 표기법: [필수 항목] [선택 항목]

설정 방법

  1. 경고 기능은 별도의 설정 없이 바로 사용할 수 있습니다.
  2. 유저 차단, 유저 추방, 역할 관리 권한이 있는 모든 유저가 사용이 가능합니다.
  3. 경고가 부여되면 지정된 로그 채널에 자동으로 기록됩니다. (로그 기능 설정 필요)
