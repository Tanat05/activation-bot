Activation Bot

더욱 강력하고 스마트한
디스코드 서버 관리

Activation은 서버 관리, 경제 시스템, 보안 기능을 하나로 통합한 올인원 디스코드 봇입니다. 복잡한 설정 없이도 서버를 활기차고 안전하게 유지할 수 있습니다.

봇 초대하기 사용 가이드

초대 및 설정

1. 봇 초대하기

Activation 봇을 서버에 초대하여 관리 권한을 부여하세요.

모든 기능을 정상적으로 사용하려면 '관리자' 권한이 필요합니다.

2. 서버 초기 셋업

/서버셋업 명령어를 사용하면 서버 운영에 필요한 필수 채널과 역할을 자동으로 생성해 드립니다.

/서버셋업

3. 로그 채널 설정

서버의 활동을 기록할 로그 채널을 한 번에 생성하거나 개별적으로 설정할 수 있습니다.

/로그채널 자동생성 - 모든 로그 채널 자동 생성 /로그채널 채팅 - 채팅 로그 채널 개별 설정

관리자 도구

서버 관리 및 설정

Admin Only
  • /서버셋업
    서버의 채널과 역할을 자동 생성합니다.
  • /서버셋업취소
    최근 5분 내에 실행한 서버 셋업을 되돌립니다.
  • /청소 [개수]
    채널의 메시지를 대량으로 삭제합니다.
  • /공지 [채널] [내용]
    깔끔한 임베드 형식의 공지사항을 전송합니다.
  • /역할지급
    유저가 버튼을 클릭해 직접 역할을 가져갈 수 있는 메시지를 생성합니다.

경제 및 활동

출석 및 포인트

매일 출석하여 포인트를 모으고 상점에서 역할을 구매하세요.

/출석 - 일일 출석체크 /역할구매 - 포인트로 역할 구매 /출석설정 포인트 - 지급 포인트 설정

채팅 레벨

채팅 활동량에 따라 레벨이 상승하며 자동으로 역할이 부여됩니다.

/채팅레벨 자동세팅 - 레벨 역할 자동 생성 /랭크 - 서버 활동 순위 확인 /유저정보 - 내 포인트 및 레벨 확인

보안 및 제재

자동 관리 및 처벌

도배, 링크, 금지 이모지 등을 자동으로 감지하고 경고 누적 시 자동으로 처벌합니다.

도배 방지
링크 차단
이모지 금지
자동 처벌

제재 명령어

  • /경고 부여 [유저] [사유] 유저에게 경고를 부여합니다.
  • /차단 [유저] [사유] 유저를 서버에서 차단합니다.
  • /자동처벌 [횟수] [처벌] 경고 누적 시 처벌 수위를 설정합니다.

신고 시스템

  • /신고 [메시지링크] 부적절한 메시지를 신고합니다.
  • /유저신고 [유저] [사유] 유저를 관리자에게 신고합니다.

유틸리티

티켓 시스템

유저가 문의를 남길 수 있는 1:1 티켓 채널을 생성합니다.

/티켓생성 - 티켓 생성 버튼 메시지 전송

자동 채널 관리

서버 인원 카운트 채널과 자동 음성 채널을 관리합니다.

/카운트채널 추가 /자동음성채널 추가

기타 기능

초대 코드별 역할 부여, 스페셜 텍스트 자동 역할 등 특별한 기능을 제공합니다.

/초대역할 추가 /스페셜텍스트 추가

관리자 도구

서버 관리 명령어

Admin Only
  • /서버설정 로그, 환영/퇴장 채널 설정
    서버의 전반적인 기능을 설정합니다. 로그 채널을 설정하면 모든 활동이 기록됩니다.
  • /청소 [개수] 메시지 대량 삭제
    채널의 메시지를 지정한 개수만큼 삭제합니다. (최대 100개)
  • /임베드 커스텀 메시지 생성
    깔끔한 임베드 메시지를 생성하여 공지사항 등을 작성합니다.
  • /이모지차단 특정 이모지 사용 제한
    서버에서 사용을 금지할 이모지를 설정합니다.

경제 및 활동

출석체크

매일 /출석 명령어로 포인트를 획득하세요. 연속 출석 시 추가 보상이 지급됩니다.

/출석 - 일일 출석 /내정보 - 포인트 및 정보 확인

레벨 시스템

채팅 활동을 통해 경험치를 얻고 레벨업하세요. 레벨에 따른 역할 자동 부여도 가능합니다.

/레벨 - 현재 레벨 확인 /랭킹 - 서버 레벨 순위

역할 상점

모은 포인트로 특별한 역할을 구매할 수 있습니다. 관리자가 상점 품목을 직접 설정합니다.

/상점 - 판매 중인 역할 목록 /구매 [역할이름] - 역할 구매

이벤트

서버에서 진행되는 다양한 이벤트에 참여하고 보상을 받으세요.

/이벤트 목록 - 진행 중인 이벤트

보안 및 제재

자동 관리 시스템

욕설, 도배, 유해 링크 등을 실시간으로 감지하고 자동으로 제재합니다. 서버의 평화를 Activation에게 맡기세요.

욕설 필터링
도배 방지
링크 차단
경고 시스템

수동 제재 명령어

  • /경고 [유저] [사유] 유저에게 경고를 부여합니다.
  • /차단 [유저] [사유] 유저를 서버에서 영구 차단합니다.
  • /추방 [유저] [사유] 유저를 서버에서 추방합니다.

처벌 기록 확인

  • /경고확인 [유저] 특정 유저의 경고 내역을 확인합니다.
  • /경고삭제 [유저] [번호] 잘못 부여된 경고를 삭제합니다.

유틸리티

티켓 시스템

1:1 문의 채널을 생성하여 체계적인 고객 지원이 가능합니다. 관리자만 접근 가능한 비공개 채널이 생성됩니다.

/티켓설정 - 티켓 시스템 활성화

숫자 세기

서버 멤버들과 함께 숫자를 세며 소통하는 미니게임을 즐기세요. 숫자가 틀리면 처음부터 다시 시작해야 합니다!

/숫자세기설정 - 채널 지정

신고 시스템

유저의 부적절한 행동을 관리자에게 익명으로 신고할 수 있습니다. 신고 내용은 지정된 로그 채널로 전송됩니다.

/신고 [유저] [내용]
